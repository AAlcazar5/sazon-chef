# Lottie Mascot Animations

Place Lottie JSON files here, named by expression:

- `excited.json`
- `thinking.json`
- `chef-kiss.json`
- `confused.json`
- `waving.json`
- `sleeping.json`
- `celebrating.json`

The `LottieMascot` component will automatically pick up any file placed here.
When a Lottie file is missing for an expression, the component falls back to the SVG `SazonMascot`.

## Design Reference

All animations should be based on the canonical mascot design: `sazon-chef-logo-orange.png` (the header mascot).
Reference SVGs are available in the parent directory for the animator.
